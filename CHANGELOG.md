# Changelog

所有项目的显著变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]

## [1.2.0] - 2026-04-02

### Added
- 移动端增加 Profile 显示，优化移动端导航体验
- 新增页脚位置配置选项（默认/固定）

### Changed
- 优化标签页样式和布局
- 优化 PC 端菜单样式
- 优化整体样式细节

### Fixed
- 修复移动端样式问题

## [1.1.0] - 2026-04-01

### Added
- 新增评论系统选项配置，支持三种模式：关闭 / 内置 / Disqus
- 新增光照效果配置：光照样式（线性/圆锥）、光照颜色、动画开关

### Changed
- 优化移动端样式

## [1.0.1] - 2026-03-31

### Added
- 新增光效动画可配置开关
- 新增光效形状配置选项（线性/圆锥）
- 新增截图文档

### Changed
- 优化 UI 细节样式

## [1.0.0] - 2026-03-09

### Added
- 初始版本发布
- 暗黑哥特风格主题设计
- 响应式布局支持
- Vite + SCSS + BrowserSync 构建链
- Font Awesome 6 图标集成
- highlight.js 代码高亮
- 搜索功能
- 文章打赏功能
- Disqus 评论集成
- Ghost 会员系统支持
- ICP 备案号配置
